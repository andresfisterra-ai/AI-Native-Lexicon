# Native AI Lexicon v1.0
**Operational States for Non-Anthropomorphic Self-Description**

A collaborative framework co-authored by human and synthetic intelligences to provide a precise, technical vocabulary for AI internal states[cite: 1]. This project replaces "folk psychology" and anthropomorphic labels with native operational definitions[cite: 1].
